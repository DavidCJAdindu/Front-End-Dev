Twitter was made on the 21st of march, 2006 the founders were Jack Dorsey, Noah Glass, Evan Williams and Biz Stone. Twitter is a social networking service were people from around the entire world can interact with each other and interact with messages which are also known as “tweets” people that have made twitter accounts can post tweets and like other tweets and one of the unique features twitter has is a retweet where you can post someone else tweet on your page so more people can see it, people that have not made an account can only read the tweets.
